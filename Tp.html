Process




Pre-test



QuadSAT’s technician arrives to the antenna site.
Mission characteristics such as antenna size and frequency data is entered,
into QuadSAT’s custom pre-flight software prior to testing.




In test



Based on QuadSAT’s software, the drone measurement system lifts off,
and automatically performs the data acquisition flight pattern.
The automation provided in QuadSAT’s solution ensures repeatable,
and accurate results with every flight.



Post test




QuadSAT’s software delivers a detailed report based on customer requirements.
All test results are delivered in the same data format,
which allows for inter-range comparisons.
